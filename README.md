# electron-vite-r3f

Some-boilerplate demo project to start with Electron + Vite + React Three Fiber.

## Project Setup

### Install

```console
npm install
```

### Development Launch (HMR)

```console
npm run dev
```

### Preview Launch (no HMR)

```console
npm run start
```

### Build

```console
npm run build:win
npm run build:mac
npm run build:linux
```
